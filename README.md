# ninja-jobs

Small playground to practice Vue Router.

## Project setup

```
yarn install
```

### Run json-server

```
npx json-server --watch data/db.json
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```
